Resume

The MMBBs project is an audio cartographic system made to understand soundscapes in collaborative work spaces, developed in the research group S’entendre (ENS Sacclay, ENSCI - les ateliers).

This system functions on the articulation of a set of devices measuring audio, distributed in space (called MMBB), and a main application (LaMAP), able to monitor these devices, record, visualize and playback the data. Moreover, this tool is intended to improve the layouts of the spaces, taking acoustic into account, and it allows to modify the maps (architecture + furnitures layout) to observe how it impacts the propagation of the sound.

Technical aspects

In this project, I’ve developped both applications, the main dataviz app in Unity3D and the C++ module’s program (using FreeRTOS for multi-threading).

LaMap also provides an interface to monitor the devices. This is essential while setting up the system.
